Test Application


The test consists of building a simple online application to show a list of cities, counties, and coordinates (latitude and longitude) in California.


View#1, must be able to view all cities, counties, and coordinates.

View#2, must be able to view all cities and coordinates in a specific county. A form will be required to a list of cities per county.


To be consistent, please import JSON data from the following URL.

  • http://www.sba.gov/about-sba-services/7617
  • http://api.sba.gov/geodata/city_links_for_state_of/CA.json

Create a simple Django project. Set up Django administration. Since Django has a built-in account management application, you can use that for authentication (login/logout).
